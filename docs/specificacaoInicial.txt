Este projeto deve usar um banco de dados SQLite e deve rodar em um docker que será instalado via app em um servidor UNRAID. Terá uma página onde será feito o upload do arquivo PDF e, uma vez os metadados extraídos e carregados no banco de dados, a página apresentará um resumo dos dados extraídos. Nenhum PDF será armazenado. Apenas os dados extraídos. Uma vez que o servidor UNRAID possui uma GPU nvidia, o projeto deve adotar um modelo de inteligência artificial apropriado para extrair os dados do PDF, inclusive analisando o conteúdo do artigo. Pesquqise no Hugging Face um modelo aberto e gratuito que possa rodar localmente e ser instalado completamente no docker que irá rodar no UNRAID, sem dependência externas. Os metadados que serão extraidos do artigo cientifico serão título, ano, autores, resumo, D.O.I., local em que foi publicado, região do estudo, comunidades envolvidas, especies de plantas identificadas, como nomes científicos e vernaculares, ano ou período em que foi conduzido os estudos, como foram coletados os dados das comunidades e o tipo de amostragem. Considerar que nem todos os artigos terão todos estes dados. O path onde o banco de dados SQLite será armazenado será passado como variável na criação do docker. Toda documentação deste projeto será gerada e mantida em português brasileiro. Este projeto terá apenas o main branch e toda a documentação e scripts será feita no main branch. Este projeto irá extrair dados de artigos científicos em PDF sobre uso de plantas em comunidades tradicionais e alimentar um banco de dados com os metadados destes artigos científicos.
